# API
## To use surbetter flask api

## RUN
---
_Init a terminal at /surfBetter/api_
### CREATE ENVIREMENT
Creat env by
```
python3 -m venv env
```
### INIT ENVIREMENT
use it
```
source env/bin/activate
```
### INSTALL REQUIREMENTS
```
(env) pip install -r apirequirements.txt
```

## RUN ENVIREMENT BY
---
## REACT
```
cd ../
npm run start-api
```
### PYTHON
```
at /surfBetter/api
env/bin/python3 main.py
```
OR
```
source env/bin/activate
(env) python3 main.py
```
